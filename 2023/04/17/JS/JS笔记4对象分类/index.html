使用new操作符

（约定俗成首字母大写,说明是用来构造对象的）

注意点2

new X()做的事情

  1. 自动创建空对象
  2. 自动为空对象关联原型，原型地址指定为 X.prototype （注意点）
  3. 自动将空对象作为 this 关键字运行构造函数
  4. 自动 return this

构造函数X

  1. X函数本身负责给对象本身添加属性
  2. X.prototype 对象负责保存对象的共用属性

原型与共用属性的关系

  1. 原型公式
    对象.__proto__===其构造函数.prototype
  2. 例外:
    Object.prototype没有构造函数
    Object.prototype.__proto__===null
  3. 原型存的是共有属性的地址

数组对象的自身属性

注意，属性名没有数字，只有字符串

数组对象的共用属性

‘push’ / ‘pop’ / ‘shift’ / ‘unshift’ / ‘join’其实就是英语

注意

  1. 函数在写的时候并没有执行，因此写函数1时，先在函数1中调用一个还未被定义的函数2，并不会报错。只有在调用函数1的时候，如果函数1调用的函数2还未被加载到内存当中，才会报错
  2. JS中所有的函数，从它存在的时候就自带一个prototype的属性，prototype中有一个constructor属性，constructor的值是它自身。也就是说function f1(){};,f1.prototype.constructor===f1的值为true
  3. new X()自动为空对象关联原型，将原型地址指定为 X.prototype。也是因此，当定义数组let arr=[1,2]，实际上是let arr=new Arrya(“1”,”2”)，因此arr的原型地址自动被指定为了Array.prototype，Object类型也是同理
  4. 属性名与属性值是两个东西
