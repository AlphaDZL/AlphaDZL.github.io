基本概念

文档流：

文档流的方向

文档中元素的流动方向。内联元素（inline元素或者说span元素）从左往右，从上往下，直到一行排满，才会另起一行。block元素（或者说div元素）从上往下，每个都另起一行。inline-block也是从左往右。

新的HTML5里，元素已不区分‘块级’和‘内联’了，所有元素都可能是块级或者内联。如果给定display:inline则为内联元素，如果给定display:block则为块级元素。

inline-block从左往右依次排开，但每个元素一定是成块的，不会出现半边在上一行，半边在下一行的情况。（inline会出现这种情况）

span元素的宽度

span（内联）元素的宽度，是由里面的所有内联元素宽度所决定的（包括文字）。（如果inline元素里，有block元素则可能被分为三行，第一行为上半截inline，第二行为block，第三行为下半截inline。所以最好不要在inline元素中写block元素）

span（内联）元素不接受用width指定宽度。

block元素的宽度

默认宽度为能有多宽，就有多宽。不影响其他元素（默认width:auto;不是100%，最好永远不要写宽度=100%)

block元素可以用width指定宽度

inline-block元素的宽度

默认宽度与inline元素相似，由里面的所有内联元素和块级元素的宽度所决定的。

但是可以像block元素一样，可以用width来指定宽度。

span元素的高度

不能用height来指定宽度，看得见的高度可以通过改变边距来改变（padding:10px 20px），但其实际高度为行高（可通过改变line-height:100px来改变）（实际行高影响span元素外层的元素的行高）

结论：inline元素的高度是由line-height间接确定，跟height无关。

具体参考：深入理解 CSS：字体度量、line-height 和 vertical-align - 知乎 (zhihu.com)

div元素的高度

由它里面的文档流元素的高度所决定。

可以用height来指定高度。

inline-block的高度

与block类似，可以设置height

脱离文档流

如果脱离文档流，则所在容器不再将该元素算在高度里

设置position: absolute;

设置float

盒模型

基本概念

CSS 盒模型是用于布局网页元素的基本概念。将每个 HTML 元素看作一个矩形盒子，并将其分为四个部分：内容区域、内边距、边框和外边距。这四个部分共同组成了一个元素的盒模型。

组成部分

CSS 盒模型中的四个部分如下：

内容区域（content）：显示元素的实际内容，如文本、图片等。

内边距（padding）：内容区域和边框之间的空白区域，用来控制元素的内部空间。

边框（border）：围绕内容区域和内边距的线条，用来界定元素的外形。

外边距（margin）：边框和相邻元素之间的空白区域，用来控制元素和其他元素之间的间距。

盒模型种类

CSS盒模型有两种实现方式：标准盒模型（content-box）和IE盒模型（border-box）。

  1. 标准盒模型（content-box）：它是CSS盒模型的标准实现方式，元素的宽度和高度只包括内容区域，不包括内边距和边框。例如，如果一个元素的宽度设置为200px，那么这个元素的内容区域的宽度就是200px。
  2. IE盒模型（border-box）：它是IE浏览器的盒模型实现方式，元素的宽度和高度包括内容区域、内边距和边框。例如，如果一个元素的宽度设置为200px，那么这个元素的总宽度就是200px，包括内容区域、内边距和边框。

图例

content-box和border-box

content-box width=内容宽度

border-box width=内容宽度+padding+border

盒模型小结

CSS盒模型分两种，一种是content-box，一种是border-box，两者区别是content-box的宽度只包含content，而border-box的宽度包含到border（由content，padding，border三部分决定）。

（border-box更加好用，这一点在同时指定padding，width和border时体现出来）

外边距合并原则

外边距会合并的情况。（外边距合并的情况只在上下边距时存在，左右外边距不会合并）

  • 父子margin合并
  • 兄弟margin合并

如果不想要合并，则写上

  • 父合并用padding/border挡住
  • 父子合并用overflow:hidden挡住
  • 父子合并用display:flex
  • 兄弟合并可以用inline-block消除

第一个子元素和最后一个子元素会出现和它们父元素外边距合并的情况。（若不想要合并，只需在父元素中加上padding，border或者overflow:hidden即可）

（如果为float元素，外边距也不会合并）

基本单位

长度单位

  1. px 像素
  2. em 相对于自身font-size的倍数
  3. 百分数
  4. 整数
  5. rem
  6. vw和vh
  7. 其他长度单位都用得很少

    颜色

  8. 十六进制#FF6600或者#F60
  9. RGBA颜色rgb(0,0,0)或者rgba(0,0,0,1)
  10. hsl颜色hsl(360,100%,100%)
