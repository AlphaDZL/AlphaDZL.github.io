JS笔记2 数据类型

发表于2023-04-15|更新于2023-05-09|JS笔记
|字数总计:1.7k|阅读时长:5分钟

前言介绍

1和’1’再存储形式上就不一样,1存的就是1,但是’1’存的是’1’的字符编码49。（当然，都是以二进制的形式）

计算机并不知道你存的是数字还是字符串，通过文件的格式来区分。比如txt里面的就是字符串，excel里面的就是数字。数字还是字符串的区别主要是在编程的时候体现出来。

JS中的七种数据类型：

  • 数字number

  • 字符串string

  • 布尔boolean

  • 符号symbol

  • 空undefined

  • 空null

  • bigint

    外加 对象Object

**需要注意的是:**数组，函数，日期并不属于数据类型，它们属于对象object。

数字number

number由64位的浮点数组成。（浮点数的意思是小数点位置可以任意动）

写法

写法 例子
整数写法 1
小数写法 0.1
科学计数法 1.23e4
八进制写法（用的少） 0123或00123或0o123
十六进制写法 0x3F 或0X3F（0x或0X开头）
二进制写法 0b11或0B11（0b或0B开头）

基本上只有前三种用的比较多。

特殊值

  1. 正0和负0
    一般情况下没有区别，区别体现在被除的时候。1/0=Infinity，1/-1=-Infinity。
  2. 无穷大
    Infinity，+Infinity，-Infinity 。
  3. 无法表示的数字
    NaN 如0/0=NaN （注：NaN===NaN的值是false）

存储结构

64位二进制存一个number

  1. 符号占1位。
  2. 指数占11位（-1023~1024）
  3. 有效数字占52位（开头的1省略）

范围

  1. 指数最大，有效数字最大，得到最大二进制数字
    Number.MAX_VALUE: 1.7976931348623157e+308
  2. 指数负方向到顶，有效数字最小1，得到最小二进制数字
    Number.MIN_VALUE: 5e+324

精度

  1. 最多只能到52+1个二进制位表示有效数字
  2. 2^53对应的十进制是9后面15个零
  3. 所以15位有效数字都能精确表示
  4. 16位有效数字如果小于等于90开头，也能精确表示
  5. 9110000000000001就存不下来

字符串string

JS中每个字符占两个字节（阉割版的UTF-8）。

写法

写法 例子
单引号 ‘你好’
双引号 “你好”
反引号 `你好`

注意: 引号并不属于字符串的一部分。

转义

用另一种写法表示想要的东西
|转义|表示|
|:—:|:—:|
|\‘|’|
|\“|”|
|\n|换行|
|\r|回车|
|\t|tab制表符|
|\||
|\uFFFF|表示对应的Unicode字符|
|\xFF|表示前256个Unicode字符|

如果想要在单引号中包含单引号。
错误的写法：’it’s ok’
正确的写法：’it\‘s ok’或”it’s ok”或`it’s ok`

字符串的属性

对象才有属性，按理来说字符串是不应该有属性的，但是由于一些渊源，字符串实际上是有属性的。

字符串的长度string.length

‘123’.length//3
需要注意的是

  1. length后面并没有括号
  2. 空字符也算一个字符
  3. 转义字符\t等算一个字符）

通过下标读取字符string[index]

let s=’hello’;s[0]//‘h’
注意：

  1. index从0开始

  2. base64编码

    （基本没用过）
  3. window.btoa(‘123’)
    正常字符串转为Base64编码的字符串
  4. window.atob(‘MTIz’)
    Bas64编码的字符串转为原来的字符串
  5. 一般用来隐藏招聘启事里的简历
    邮箱：ZmFuZ3lpbmdoYW5nQGZveG1haWwuY29t
  6. 有时候也用来自欺欺人
    所谓的「加密」，也就能骗过一部分外行

布尔值

得到bool值的运算符

以下运算符会得到bool值

  • 否定运算 !value
  • 相等运算 1==2, 1!=2, 3===4, 3!==4
  • 比较运算 1<2 1>2

if配bool

if语句通常都需要判断真假
if(value){}else{}

value是bool值的话不必多言，那不是bool值的情况，1和0，’1’和’0’，谁真谁假呢？因此引入5个falsy值。

五个falsy值

falsy就是相当于false，但又不是false的值。
五个falsy值分别是：1. undefined、2. null、3. 0、4. NaN 、5. ‘’(空字符串)。

注意：

  1. 0是假，’’是假，但是’0’不是假，而是真，因为它是非空字符串。
  2. 除了这五个falsy以及false外，JS里其他的值都为true。
  3. 注意’’和’ ‘的区别，前者（空字符串）相当于false，后者（非空字符串）相当于true。

undefined & null

两种空类型。这两种空并没有本质的区别。
有区别的情况：

  1. 如果一个变量声明了，但没有赋值，那么默认值就是undefined，而不是null。
  2. 如果一个函数，没有写return，那么默认return就是undefined，而不是null。
  3. undefined更多的是一个默认的空，null更多是一个主动的空。

symbol

用到的情况几乎没有。详细内容参考MDN Symbol

bigint

BigInt是一种内置对象，它提供了一种方法来表示大于2^53 - 1的整数，可以表示任意大的整数。详情参考MDN BigInt

变量声明

三种声明方式
|声明方式|区别|
|:—:|:—:|
|var a=1|var是过时的、不好用的方式|
|let a 1|let是新的，更合理的方式|
|const a 1|const是声明时必须赋值，且不能再改的方式|
|a=1|这种方式是错误的，不要这样声明|

var变量声明之后写篇文章单独展开。

let声明

特点

  1. 遵循块作用域，即使用范围不能超出{}
  2. 不能重复申明（不同作用域可以再次声明）（var可以重复声明）
  3. 声明的时候可以赋值，也可以不赋值
  4. 必须先声明再使用，否则报错
  5. 全局声明的let变量（在最顶层的作用域），不会变成window的属性（var会）
  6. for循环配合let有奇效

const声明

只读变量，再次修改的时候会报错。

特点（与let相似）

  1. 遵循块作用域，使用范围不能超出{}
  2. 不能重复申明
  3. 声明的时候必须赋值
  4. 作用域可以是全局或本地声明的块，全局常量不会变为 window 对象的属性

注意

变量声明在声明的同时也指定了类型（undefined也是一种类型），但是值和类型都可以随意变化。

类型转换

  1. number=>string
    String(n)或n+’’
  2. string=>number
    Number(s)或parseInt(s)或parseFloat(s)或s-0或+s
  3. x=>bool（x表示任何类型）
    Boolean(x)或!!x
  4. x=>string
    String(x)或x.toString()
    此处JS有个bug。true.toString()会输出”true”，1.01.toString()会输出”1.01”，但是1.toString()会报错，因为当一个数字类型连着一个.时，JS会把.当成是数字的一部分，当JS读到1.t的时候就已经报错了。为解决这个bug，可写成(1).toString()或者1..toString()或者1 .toString()

JS中可能遇到的各种奇怪事情，可以搜索JS 秘密花园参考。
